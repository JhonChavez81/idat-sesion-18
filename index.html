Aprende a maquetar para todos los dispositivos usando "Mobile First" y CSS moderno.

El concepto "Mobile First"

En lugar de diseñar para escritorio y "encoger" el contenido, diseñamos primero para el teléfono y expandimos.

  • Rendimiento: Los dispositivos móviles descargan menos código complejo.
  • Priorización: Nos obliga a mostrar solo lo más importante.
  • Escalabilidad: Es más fácil añadir columnas que quitarlas.

Media Queries (@media)

Son condiciones lógicas en CSS. Si la pantalla cumple la condición, se aplican los estilos internos.

Regla de oro: En Mobile First, usamos principalmente min-width. 
/* 1. Estilos por defecto (MÓVIL) */
.container {
  display: flex;
  flex-direction: column;
}

/* 2. Cuando la pantalla es >= 768px (TABLET/PC) */
@media (min-width: 768px) {
  .container {
    flex-direction: row;
  }
}

Taller: Galería Grid (Código)

HTML

<div class="galeria">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>

CSS (El Truco)

/* Para móvil (1 columna) */
.galeria {
  display: grid;
  grid-template-columns: 1fr;
  gap: 10px;
}

/* Para PC (3 columnas) */
@media (min-width: 1024px) {
  .galeria {
    grid-template-columns: repeat(3, 1fr);
  }
}

Animaciones Nativas

CSS nos permite crear micro-interacciones sin Javascript usando transition y hover.

.caja {
  transition: transform 0.3s ease;
}

.caja:hover {
  transform: scale(1.2) rotate(15deg);
}

Pauta Final

Etiqueta Viewport

Nunca olvides agregar el <meta name="viewport"> en tu HTML. Sin él, el responsive no funciona.

Unidades Relativas

Evita anchos fijos (width: 500px). Usa porcentajes (%) o anchos máximos (max-width: 100%).

Imágenes Flexibles

Tus imágenes siempre deben llevar max-width: 100%; height: auto; para no romper el layout.
